# zh-official-lab

本仓库用于归档、托管与发布多个独立 HTML 页面。这些页面以静态形式存放，不依赖后端服务，可直接通过浏览器访问。

## 仓库说明

- 仓库中的 HTML 文件均为独立页面，每个文件对应一个功能或展示单元。
- 文件之间无强依赖关系，可单独引用或部署。
- 页面内容可能涉及信息展示、工具界面或实验性演示，不针对任何特定域名或网站。

## 目录结构

```
.
├── index.html          # 首页 / 索引页（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 每个 HTML 页面建议自包含样式与脚本，或引用 `assets/` 目录下的公共资源。
- 页面文件名应简洁且具有描述性，方便识别用途。
- 如需更新或替换页面，请直接修改对应文件并提交 Pull Request。

## 维护说明

- 本仓库由维护者不定期更新，欢迎提交 Issue 或 Pull Request。
- 提交新页面时，请确保文件结构清晰，不引入外部不可控资源。
- 请勿在页面中添加跟踪代码、广告或诱导性内容。
